# pdb_creator_python
## Hot to compile:

1. move the "pkg_dec.exe" in the same folder with the pdb_creator.py
2. install the dependencies:
	- pip install wget
	- pip install pyinstaller
3. double click the build.bat in the "scripts"-Folder
